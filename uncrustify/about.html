About the Uncrustify Documentation Project

This Uncrustify Documentation Project is not the official uncrustify documentation.
In fact uncrustify doesn't provide a real documentation. 

> uncrustify --help
  Usage:
  uncrustify [options] [files ...]
  ...
  There are currently 850 options and minimal documentation.
  Try UniversalIndentGUI and good luck.
(Code here)

Because uncrustify users are developers, uncrustify source code is too often used as the official documentation.
The best you can get (outside the code) is this documentation directory. The wiki is only about the internal design of uncrustify.

The Uncrustify Documentation Project is an independant effort to make more obvious the effect of each of the ~850 options.
