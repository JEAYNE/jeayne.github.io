About this uncrustify documentation

This project is not the official uncrustify documentation. In fact uncrustify doesn't provide a real documentation. 

> uncrustify --help
  Usage:
  uncrustify [options] [files ...]
  ...
  There are currently 850 options and minimal documentation.
  Try UniversalIndentGUI and good luck.
(Code here)

Because uncrustify users are developers, uncrustify source code is too often used as the official documentation.
The best you can get (outside the code) is this documentation directory. The wiki is only about the internal design of uncrustify.

This project is an independant effort to make more obvious the effect of each uncrustify options.

  • Report problems related to this documentation here.
  • This documentation project is not managed by the uncrustify team.
